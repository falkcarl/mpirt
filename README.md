# mpirt

## Overview

mpirt contains a collection of functions and wrappers for estimating
monotonic polynomial item response models using *rpf* and *OpenMx*.

## Installation

``` r
# From GitHub:
# install.packages("devtools")
devtools::install_github("falkcarl/mpirt")
```
